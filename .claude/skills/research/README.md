# Source Verifier

Verify sources and citations when researching. Use when you need to
fact-check claims, validate sources, or ensure research accuracy by
examining actual source content rather than summaries or snippets.

## Structure

- `SKILL.md` - Main skill instructions
- `references/` - Detailed documentation loaded as needed
- `scripts/` - Executable code for deterministic operations
- `assets/` - Templates, images, or other resources
